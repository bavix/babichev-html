# grpctestify

grpctestify — утилита для тестирования gRPC серверов.

## Описание

Это утилита для тестирования GripMock. Был создан, чтобы облегчить жизнь и улучшить coverage сервиса.

## Установка

```bash
# Download the latest release
curl -LO https://github.com/gripmock/grpctestify/releases/latest/download/grpctestify.sh
chmod +x grpctestify.sh

# Verify installation
./grpctestify.sh --version
```
