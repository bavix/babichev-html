# Laravel Wallet

Laravel Wallet — пакет для работы с виртуальными кошельками в Laravel приложениях.

## Описание

Пакет предоставляет функциональность виртуального кошелька для управления балансами пользователей, проведения транзакций и отслеживания истории операций.

## Основные возможности

- Мультивалютная поддержка
- Управление транзакциями
- Переводы между кошельками
- Расширяемая архитектура

## Установка

```bash
composer require bavix/laravel-wallet
```
